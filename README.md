# Awesome CLI Tool

A simple and lightweight CLI tool to automate repetitive tasks.

This project is hosted on GitHub.

## 🚀 Features

* Easy-to-use command line interface
* Fast execution
* Extensible structure
* No unnecessary dependencies

## 📦 Installation

```bash id="install01"
git clone https://github.com/qianyuyn/awesome-cli-tool.git
cd awesome-cli-tool
```

## ▶️ Usage

```bash id="usage01"
python main.py --help
```

Example:

```bash id="usage02"
python main.py run --task greet
```
